## ğŸ¨ Design Principles

### Color Palette
- **Primary Gradient**: Purple to violet (`#667eea` â†’ `#764ba2`)
- **Accent**: Cyan (`#64ffda`)
- **Background**: Deep navy (`#0a0e27`, `#0f1429`)
- **Text**: White and light gray for optimal readability

### Typography
- **Headings**: Inter (weights 700-900)
- **Body**: Inter (weights 300-600)
- **Code**: JetBrains Mono

### Glass Effect
- Background: `rgba(255, 255, 255, 0.05)`
- Blur: `20px` backdrop filter
- Border: `1px solid rgba(255, 255, 255, 0.1)`
- Shadow: `0 8px 32px rgba(0, 0, 0, 0.3)`

## ğŸš€ Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/WasifArmanHaque/wasifarmanhaque.github.io.git
   cd wasifarmanhaque.github.io
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server (recommended)
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **Make changes**
   - Edit `index.html` for content
   - Edit `css/glassmorphism.css` for styling
   - Edit `js/glassmorphism.js` for interactions

### Customization

#### Update Personal Information
Edit `index.html` to update:
- Name and title
- About section content
- Experience timeline
- Skills and technologies
- Contact information

#### Modify Colors
Edit CSS variables in `css/glassmorphism.css`:
```css
:root {
  --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --accent-cyan: #64ffda;
  --bg-dark: #0a0e27;
  /* ... more variables */
}
```

#### Adjust Animations
Modify animation settings in `js/glassmorphism.js`:
- Fade-in timing
- Parallax effects
- Scroll behaviors
